    Conceito de Clientes
    Design Responsivo ou responsividade

    A responsividade garante que a página web se adeque aos tamanhos de telas, sejam elas grandes (Desktop), ou em dispositivos móveis (mobile)

    Navegadores

    Cabeçalhos são normalmente utilizados para identificar páginas e seções e possuem aparência diferenciada do restante do texto.

    • São programas criadas por empresas, utilizados para abrir/executar arquivos.
    • Seguem padrões W3C, porém, sempre tem uma diferença ou outra de interpretação.
    • Também chamados de “browsers”
    • Iniciou com o MOSAIC, passou para o Netscape, e hoje temos uma variedade de navegadores disponíveis.

    Aplicações web

    Aplicações Web são soluções criadas que possuem a internet como meio de comunicação entre Client x Server não sendo necessário a sua instalação. Essas aplicações estão interligados entre si, e elas fornecem a essa comunicação entre cliente e servidores.

    Estrutura básica do HTML

    HTML (abreviação de Hypertext Markup Language) é uma linguagem de marcação utilizada na estruturação de páginas web. Sua sintaxe é bastante simples e, assim como a XML é baseada em tags, que representam os diversos elementos de uma página, como imagens e links.

    A estrutura básica de uma página HTML pode ser vista na Listagem 1, na qual podemos ver as principais tags que são necessárias para que o documento seja corretamente interpretado pelos browsers.

    Lista ordenada e não ordenada

    1. ordenadas são aquelas que seguem uma ordem crescente.
    • Não ordenadas significa que não precisa de uma ordem, ou seja apenas de marcadores para serem identificadas como uma lista de itens por exemplo.
